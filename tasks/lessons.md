---

## 디자인 규칙
- **컬러 팔레트 준수**: 임의 색상(`gray-500`, `blue-600` 등 Tailwind 기본값) 절대 사용 금지. 반드시 CLAUDE.md의 DEARLOG Warm Palette 사용
- **폰트**: Noto Sans KR 외 다른 폰트 사용 금지
- **모바일 우선**: max-width 390px 기준. 데스크탑 레이아웃 먼저 만들지 말 것
- **시니어 UI**: 글씨 최소 16px, 버튼 최소 48px 높이 — 작은 요소 만들면 안 됨

---

## 코딩 규칙
- **타입**: 새 타입은 반드시 `src/types/`에 정의 후 import. 컴포넌트 내 inline 타입 정의 금지
- **컴포넌트 파일명**: PascalCase (`InterviewCard.tsx`, not `interviewCard.tsx`)
- **상태관리**: 로컬 UI 상태 외 모든 앱 상태는 Zustand 스토어 사용
- **주석**: 비자명한 WHY만 작성. WHAT 설명 주석 금지

---

## AI 에이전트 규칙
- **원문 보존 우선**: AI는 보관자. 사용자가 말한 내용을 절대 창작·변형하지 않음
- **에이전트 분리**: 에이전트 하나당 하나의 역할만 부여
